## 🌐 API Information

### Current API
**Frankfurter API**
- Free, no authentication required
- Real-time European Central Bank rates
- 180+ currencies supported
- No rate limits for development
- CORS enabled
- URL: `https://api.frankfurter.app/latest?from=USD`

### Offline Support
- Automatically caches rates in LocalStorage
- Uses cached rates if offline
- Shows "offline" indicator
- Auto-fetches when back online

### Alternative APIs (If Needed)
- ExchangeRate-API (requires free key)
- Exchange.host (free, no key)
- Open Exchange Rates (free tier available)
- Easy to switch with simple code change

## 💾 Data Persistence

### What Gets Saved
1. **Exchange Rates** - Cached for offline use
2. **Last Update Time** - When rates were last fetched
3. **Conversion History** - Last 50 conversions
4. **Favorites** - Marked currency pairs
5. **Theme** - Light/Dark mode preference

### Storage Location
- Browser's LocalStorage
- ~5MB available per domain
- Persists across browser restarts
- Can be cleared by user

### Data Structure
```javascript
{
  "currentRates": { ... },
  "lastUpdated": "2024-06-13T...",
  "history": [ ... ],
  "favorites": [ ... ]
}
```
